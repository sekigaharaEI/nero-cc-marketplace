# 飞书消息发送 Skill - 原始需求

## 背景

创建一个 Claude Code 插件,提供通过飞书发送通知的能力。

## 核心需求

创建一个插件,包含两个 skills:

### 1. feishu-send-notification (发送通知)
- **功能**: 教模型如何向飞书发送消息
- **支持**: 个人对话和群组对话的单向通知
- **调用方式**: 模型可以根据需要自动调用

### 2. feishu-setup (安装引导)
- **功能**: 引导用户如何安装和配置飞书发送服务
- **配置项**: disable-model-invocation: true (仅用户手动调用)
- **内容**: 
  - 服务安装步骤
  - 飞书应用配置指南
  - 环境变量配置
  - 测试验证方法

## 技术要求

- 单向通知(只发送,不接收)
- 支持文本消息
- 使用飞书开放平台 HTTP API
- 配置简单明了
