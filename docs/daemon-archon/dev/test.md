## 测试命令参考

```bash
# 环境检查
/archon-init

# 启动服务
/archon-start

# 查看状态
/archon-status

# 创建 Probe 任务
/start-probe <任务描述>

# 创建 Cron 任务
/start-cron <任务描述>

# 检查任务
/check-task <task_id>

# 查看任务列表
curl -s http://127.0.0.1:8765/tasks | python3 -m json.tool

# 查看任务详情
curl -s http://127.0.0.1:8765/tasks/<task_id> | python3 -m json.tool

# 查看服务日志
tail -f ~/.claude/daemon-archon/server.log

# 查看任务日志
tail -f ~/.claude/daemon-archon/<task_id>/archon.log
```
