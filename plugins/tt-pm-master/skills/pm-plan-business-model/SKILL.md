# SKILL: 商业模式规划

## 元信息

- **版本**: 1.0
- **适用场景**: 新产品/新业务的商业模式规划，面向高管汇报
- **输出物**: 商业模式规划文档（Markdown格式）
- **参考案例**: 金太阳数字化平台商业模式规划

---

## 触发条件

当用户需要以下任务时触发此SKILL：

- "帮我做商业模式规划"
- "设计一个新产品的商业模式"
- "写一份商业计划书"
- "规划新业务的盈利模式"

## 输入要求

### 必需信息（缺失时主动追问）

| 信息类型     | 说明                           | 追问示例                                 |
| :----------- | :----------------------------- | :--------------------------------------- |
| **业务背景** | 公司现有业务、资源、优势       | "请介绍一下公司现有的核心资源和业务情况" |
| **目标市场** | 目标用户群体、市场规模         | "目标用户是谁？市场规模大概多大？"       |
| **竞品信息** | 主要竞争对手及其模式           | "主要竞品有哪些？有竞品截图或资料吗？"   |
| **汇报对象** | 文档读者（总经理/投资人/团队） | "这份文档是给谁看的？"                   |

### 可选信息（有则更好）

- 竞品APP截图
- 内部财务数据
- 已有用户数据
- 技术团队配置
- 预算范围

## 执行流程

### Phase 1: 信息收集与竞品分析

```
1. 收集竞品信息
   ├── 竞品APP截图分析（如有）
   ├── 竞品商业模式拆解
   ├── 竞品定价策略
   └── 竞品优劣势分析

2. 盘点自身资源
   ├── 核心资产（内容、用户、渠道）
   ├── 团队能力
   ├── 资金预算
   └── 现有业务协同点

3. 输出：竞品商业模式分析报告
```

### Phase 2: 商业模式设计

```
1. 确定商业模式类型
   ├── 免费增值（Freemium）
   ├── 订阅制（Subscription）
   ├── 交易抽成（Marketplace）
   ├── 广告变现（Advertising）
   └── 混合模式（Hybrid）

2. 设计核心变现路径
   ├── 流量入口（如何获客）
   ├── 价值交付（提供什么价值）
   ├── 付费转化（如何变现）
   └── 用户留存（如何提升粘性）

3. 定价策略设计
   ├── 参考竞品定价
   ├── 设计价格梯度
   └── 设计促销策略
```

### Phase 3: 财务预测

```
1. 成本估算
   ├── 研发成本
   ├── 运营成本
   ├── 获客成本
   └── 内容/服务成本

2. 收入预测
   ├── 用户增长预测
   ├── 转化率假设
   ├── ARPU估算
   └── 分阶段收入预期

3. 关键假设说明
   ├── 列出核心假设
   ├── 设定验证指标
   └── 设定达标线/预警线
```

### Phase 4: 风险分析与决策建议

```
1. 风险识别
   ├── 市场风险
   ├── 政策风险
   ├── 运营风险
   ├── 技术风险
   └── 转化风险

2. 应对措施
   ├── 每个风险的应对策略
   └── 止损机制设计

3. 决策建议
   ├── 核心决策点
   ├── 需要领导决策的事项
   └── 下一步行动计划
```

## 常用分析框架

### 商业模式画布（简化版）

| 要素         | 问题                     |
| :----------- | :----------------------- |
| **价值主张** | 我们为用户解决什么问题？ |
| **目标客户** | 谁是我们的用户？         |
| **渠道通路** | 如何触达用户？           |
| **收入来源** | 如何赚钱？               |
| **成本结构** | 主要成本是什么？         |
| **核心资源** | 我们有什么独特资源？     |
| **关键活动** | 我们需要做什么？         |

### 竞品分析维度

| 维度           | 分析内容     |
| :------------- | :----------- |
| **流量来源**   | 如何获客     |
| **核心变现**   | 主要收入来源 |
| **定价策略**   | 价格区间     |
| **边际成本**   | 规模化后成本 |
| **运营复杂度** | 团队要求     |
| **政策风险**   | 监管影响     |

### 财务预测公式

```
年收入 = 用户数 × 付费转化率 × ARPU

其中：
- 用户数 = 获客渠道触达量 × 下载转化率 × 注册激活率
- 付费转化率 = 付费用户数 / 注册用户数
- ARPU = 年均付费金额
```

---

## 注意事项

1. **避免过度乐观**：财务预测使用保守估算，标注乐观/基准/保守场景
2. **突出差异化**：明确说明"我们有什么竞品没有的"
3. **量化风险**：风险分析要有概率、影响、验证指标
4. **可执行性**：决策建议要具体到"本周内做什么"
5. **精简正文**：详细方案移至附录，正文控制在800行以内
