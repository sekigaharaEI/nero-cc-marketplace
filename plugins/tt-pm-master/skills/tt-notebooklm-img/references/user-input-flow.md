# 用户输入收集流程

**必须按顺序逐步询问，每个步骤确认完再进入下一步。**

## 步骤0：选择模式

```json
{
  "questions": [{
    "question": "请选择图片生成模式",
    "header": "生成模式",
    "options": [
      {"label": "文生图", "description": "用文字描述生成全新图片"},
      {"label": "改图", "description": "上传原图 + 修改意见，在原图基础上修改"},
      {"label": "混合模式", "description": "部分文生图 + 部分改图"},
      {"label": "批量导入", "description": "已有 picture-set.md 文件，直接上传生成"}
    ],
    "multiSelect": false
  }]
}
```

### 选择"批量导入"时

扫描当前目录下的 `.md` 文件：

```
**/*.md
```

展示文件列表后让用户选择：

```json
{
  "questions": [{
    "question": "请选择已有的 picture-set.md 文件",
    "header": "选择文件",
    "options": [
      {"label": "picture-set.md", "description": "从目录中找到的文件"},
      {"label": "其他文件.md", "description": "从目录中找到的文件"}
    ],
    "multiSelect": false
  }]
}
```

选择文件后：
1. 读取文件内容，解析出图片数量
2. 检查是否有"参照原图"关键字，如有则扫描目录中的图片文件，确认原图是否存在
3. 展示摘要信息给用户确认
4. **直接跳到步骤4（自动化执行），跳过步骤3（构建 picture-set.md）**

---

## 步骤1：收集图片描述

### 文生图模式

直接在对话中询问：

```
请描述你想生成的图片。你可以：

1. 一次描述多张图片（用序号分隔）
2. 每张图片建议包含：风格、内容描述
3. 最多支持 20 张

示例格式：
第1张：赛博朋克风格，霓虹灯照耀下的雨夜街道...
第2张：水彩风格，春天的樱花树下一只猫在睡觉...

请输入你的图片描述：
```

**Claude 的处理逻辑**：
- 如果用户只给了简短描述（如"画一只猫"），Claude 应主动扩写为详细的图片描述（50-150字），然后展示给用户确认
- 如果用户给了详细描述，直接使用
- 如果用户一次描述了多张，逐一确认

### 改图模式

先扫描当前目录下的图片文件：

```
**/*.{jpg,jpeg,png,gif,webp}
```

展示文件列表后，使用多选让用户选择原图：

```json
{
  "questions": [{
    "question": "请选择要修改的原图（可多选）\n\n选择 Other 时请输入文件路径",
    "header": "选择原图",
    "options": [
      {"label": "photo1.jpg", "description": "图片文件"},
      {"label": "photo2.png", "description": "图片文件"}
    ],
    "multiSelect": true
  }]
}
```

选择原图后，**逐张在对话中询问修改意见**：

```
原图：photo1.jpg
请输入修改意见（例如："把背景改成星空"、"添加一只猫在左下角"）：
```

### 混合模式

先询问总共几张图片：

```
请问总共需要生成几张图片？（最多20张）
```

然后对每张图片：

```json
{
  "questions": [{
    "question": "第 N 张图片的类型",
    "header": "第N张",
    "options": [
      {"label": "文生图", "description": "用文字描述生成新图片"},
      {"label": "改图", "description": "基于原图修改"}
    ],
    "multiSelect": false
  }]
}
```

根据选择进入对应的收集流程。

---

## 步骤2：确认所有信息

**重要：此步骤禁止使用 AskUserQuestion 弹窗！**

收集完所有信息后，直接在对话中展示确认信息：

```
即将生成图片，请确认以下信息：

| 序号 | 类型 | 描述摘要 |
|------|------|----------|
| 第1张 | 文生图 | 赛博朋克风格，霓虹灯雨夜街道... |
| 第2张 | 改图 | 原图: photo1.jpg，修改: 把背景改成星空 |
| 第3张 | 文生图 | 水彩风格，春天樱花树下的猫... |

共 3 张图片。

请回复"确认"开始生成，或告诉我需要修改的地方：
```

等待用户确认后，进入步骤3（构建 picture-set.md）。
